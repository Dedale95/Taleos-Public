Télécharger l'extension

⬇ Télécharger le ZIP
✅ L'extension se recharge…
1
Téléchargez le ZIP ci-dessus et extrayez-le dans le dossier de votre choix.
2
Ouvrez Chrome et allez sur chrome://extensions. Activez le Mode développeur (interrupteur en haut à droite).
3
Cliquez sur « Charger l'extension non empaquetée » et sélectionnez le dossier extrait.
4
L'extension Taleos apparaît dans votre barre d'outils. Connectez-vous via la popup pour démarrer.
💡 Pour les mises à jour suivantes : remplacez les fichiers du dossier, puis cliquez sur 🔄 Recharger l'extension (si l'extension est déjà installée) ou utilisez le bouton « Mettre à jour » dans la popup Taleos.
